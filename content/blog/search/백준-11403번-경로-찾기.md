- https://www.acmicpc.net/problem/11403

## Solution

- 1차 시도: 실패

```py
import sys
from collections import deque

input = sys.stdin.readline

n = int(input())

graph = [list(map(int, input().split())) for _ in range(n)]


def bfs(x):
    visit = [0] * n
    queue = deque([x])

    while queue:
        v = queue.popleft()
        for i in range(n):
            if graph[v][i] == 1 and not visit[i]:
                queue.append(i)
                visit[i] = 1
    for i in range(n):
        print(visit[i], end=' ')
    print()


for i in range(n):
    bfs(i)
```
